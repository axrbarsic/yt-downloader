# YT Downloader

Мобильное веб-приложение для скачивания видео с YouTube. Работает в Safari и Chrome на iPhone/Android.

## Как использовать

1. Открой сайт: `https://axrbarsic.github.io/yt-downloader/`
2. Вставь ссылку на YouTube-видео
3. Выбери качество (720p, 1080p, MP3)
4. Нажми «Скачать»

## Как это работает

- Фронтенд: статический HTML (GitHub Pages)
- Бэкенд: FastAPI + yt-dlp (получает прямые ссылки с YouTube)
